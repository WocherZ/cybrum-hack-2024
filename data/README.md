# Датасеты

- **_train_dataset_advertisement.jsonl_**: обучающий датасет, сгенерированный сильными языковыми моделями.
При генерации использовались как инструкции для генерации рекламы на общие темы, так и инструкции для генерации рекламы на тематику цифровых продуктов Росатома.
- **_validation_advertisement_dataset.jsonl_**: тестовый датасет, для оценки моделей, созданный по тематике цифровых продуктов Росатома.
